# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project using **Python, EDA, PostgreSQL, SQL, and Power BI** to analyze customer shopping behavior and generate actionable business insights.

##  Project Overview

This project analyzes **3,900 retail transactions** to uncover customer purchasing patterns, product performance, subscription behavior, and revenue trends. The workflow includes data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and an interactive Power BI dashboard.

##  Objectives

* Clean and prepare raw retail data for analysis
* Perform Exploratory Data Analysis (EDA)
* Answer business questions using SQL
* Build an interactive Power BI dashboard
* Generate actionable business insights

##  Tech Stack

* **Python** (Pandas, NumPy)
* **Exploratory Data Analysis (EDA)**
* **PostgreSQL**
* **SQL**
* **Power BI**

##  Project Workflow

```

      ↓

      ↓

      ↓
```

##  SQL Business Analysis

The following business questions were solved using SQL:

1. Revenue generated by male vs. female customers
2. High-spending customers who used discounts
3. Top 5 highest-rated products
4. Average purchase amount by shipping type
5. Subscriber vs. non-subscriber spending analysis
6. Products with the highest discounted purchases
7. Customer segmentation (New, Returning, Loyal)
8. Top 3 purchased products in each category
9. Repeat buyers and subscription analysis
10. Revenue contribution by age group

##  Dashboard Highlights

* KPI Cards (Revenue, Orders, Average Purchase, Ratings)
* Revenue Analysis
* Customer Demographics
* Product Performance
* Customer Segmentation
* Subscription Analysis
* Interactive Filters & Slicers

##  Key Insights

* Subscribers contributed higher overall revenue.
* Loyal customers generated a significant share of sales.
* Certain products relied heavily on discounts.
* Express shipping customers spent more on average.
* Revenue varied across different customer age groups.
